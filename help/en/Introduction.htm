vnforest Data sharing system

Introduction

Data Sharing System is a tool for browsing the forest resource data. It enables display of maps of various forestry themes, querying the information of a forestry plot, overlaying forestry data with various background maps and and querying statistics of selected plots.

The user interface components are presented in the picture below.

1.       The Title Bar

The name of the application, Forestry Data Sharing System is displayed on the Title Bar

The name of the curretly selected forestry theme is displayed on the title bar

3.       Language selection

You can select to run the application in English or in Vietnamese

4.      Menu bar

The menus are explained separately

5.       Map window

The map window presents the overlay of the background map, the selected forestry map and the selected reference data.

6.      Map legend window

Map legend window has three tabs.

       Map legend tab displays the legend of the currently selected forestry map

       Map layers tab presents the currently selected background map, forestry map and reference map layers

       Info tab displays results of an attribute query

7.       Zoom in and Zoom out buttons

The zoom buttons help you to change the map scale. You can also use the scroll wheel of the mouse to zoom in and out. And you can move the map by dragging with the mouse (clicking on the map, keeping the mouse button down and moving the mouse).

8.      Scale bar

The scale bar shows a line and displays its length.

The number below the scale bar indicates the current zoom level. The application works with zoom levels 6-18.

       In the smallest zoom level (6) the map is zoomed around the whole Vietnam

       In the largest zoom level (18) the map is zoomed to a couple of plots

       The forestry maps can be displayed at zoom levels 13-18.
