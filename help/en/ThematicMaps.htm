Thematic maps

The Thematic Maps menu includes the main forestry themes that you can display on the map.

       Only one forestry map is active in the display. Selecting one map theme automatically deselects the previous one.

       The forestry maps display only when you have zoomed close enough, to map scale larger than 1:70.000 (i.e. to district or commune level, depending on the size of the district). In smaller scales the map data would not be suitable for viewing. You can select a map also in smaller scales. Then the application will warn about unsuitable scale for the forestry maps. You can zoom closer in order to display the forestry map.

On the list laid out through the menu you need to first open one of the main themes (Forest resource data 2016 / National Forest Inventory and Statisics (NFIS) / / ) in order to select display of one of the forestry maps under that category.

Picture: Example of opening first the Forest resource data 2016 main theme and then selecting the Forest type map.

The name of the active theme will be displayed as the map title on the top bar.

Your selection will also change the content of the Map Layers and Map Legend windows.

The Map layers, Map legend and Info forms are alternative. You can toggle between these sheets by clicking at the header of the desired sheet. Yellow color indicates the active sheet.

Picture: The map legend window has three tabs. Yellow colour indicates which one is active.

The opacity slider will change the opacity of the forestry map. If you have selected Relief or Hillshading as the background map you can get an idea of the topography in relation with the forestry information.

Picture: Relief selected as the background map and some opacity set for the forestry map.
